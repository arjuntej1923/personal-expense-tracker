# 💰 Personal Expense Tracker (Console-Based Application)

A menu-driven Personal Expense Tracker built using **Python**. This application helps users efficiently manage their daily expenses through a simple Command-Line Interface (CLI). It supports complete CRUD operations, data persistence using CSV files, input validation, and summary analytics.

---

## 📌 Features

- ✅ Add New Expense
- ✅ View All Expenses (sorted by date)
- ✅ Search Expense by ID
- ✅ Update Existing Expense
- ✅ Delete Expense
- ✅ View Financial Summary
  - Total Expenses
  - Average Expense
  - Highest Expense
  - Lowest Expense
  - Category-wise Breakdown
- ✅ Search Expenses by Category *(Bonus Feature)*
- ✅ Monthly Expense Summary *(Bonus Feature)*
- ✅ Auto-generated Expense IDs
- ✅ Automatic Date Assignment
- ✅ CSV File Storage for Data Persistence
- ✅ Robust Input Validation
- ✅ Professional Tabular Output

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/arjuntej1923/personal-expense-tracker.git
```

2. Navigate to the project folder

```bash
cd personal-expense-tracker
```

3. Run the application

```bash
python expense_modified.py
```

```
1. Add New Expense
2. View All Expenses

4. Update Expense
5. Delete Expense
6. View Summary Metrics
7. Search by Category
8. Monthly Summary
9. Exit & Save Data
```

---

## 💾 Data Persistence

- Expense records are stored in a CSV file.
- Existing data is automatically loaded when the application starts.
- Changes are automatically saved after every Add, Update, and Delete operation.
- Data is also saved safely when exiting the application.
